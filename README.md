Setup Instructions
Prerequisites
Java Development Kit (JDK): Ensure you have JDK 17 or higher installed.

NetBeans IDE: Open the project directly in NetBeans to maintain library references.

How to Build & Run
Open the project in NetBeans.

Right-click the project name in the sidebar.

Select "Clean and Build".

Once the build succeeds, click the "Run" button in the toolbar.
